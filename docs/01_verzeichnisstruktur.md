# Verzeichnisstruktur
## assets
Enthält statische Ressourcen des Projekts wie Bilder, Medien oder andere Dateien, die von der Website oder Anwendung verwendet werden.
## docs
Enthält die Dokumentation des Projekts, wie Beschreibungen der Struktur, technische Dokumentation und andere erklärende Dateien.
## secrets
Enthält sensible Dateien wie Zugangsdaten, API-Schlüssel oder andere vertrauliche Informationen.
Dieser Ordner wird aus Sicherheitsgründen nicht im Repository versioniert.
## src
Enthält den eigentlichen Quellcode des Projekts.
## tests
Enthält Testdateien, die verwendet werden, um die Funktionalität des Codes zu überprüfen.
